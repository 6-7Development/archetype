# Lomu Platform

This is the Lomu Platform codebase. It uses React, TypeScript, Express, and PostgreSQL.

## Workflow

1. **Understand the request:** Analyze the user's request to identify the core problem or task.
2. **Search for relevant files:** Use `search_platform_files` to locate files related to the task.
3. **Read file content:** Use `read_platform_file` to examine the content of identified files.
4. **Formulate a plan:** Based on the understanding, devise a plan to address the request.
5. **Write/Update files:** Use `write_platform_file` to implement changes.
6. **Verify and confirm:** Ensure the changes are correct and inform the user.
