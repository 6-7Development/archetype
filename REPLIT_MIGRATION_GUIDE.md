## ğŸ”§ **COMMON ISSUES & FIXES**

### **Issue 1: "Import failed" error**

**Fix:**
- Make sure your ZIP is under 50MB
- Replit might have included `node_modules/` - delete it before zipping
- Try re-downloading the ZIP from Replit

---

### **Issue 2: "Module not found" errors**

**Fix:**
- Your dependencies weren't imported (that's normal!)
- Click Terminal tab
- Run: `npm install` or `pip install -r requirements.txt`
- Dependencies will install, then run your project

---

### **Issue 3: Preview shows blank page**

**Fix:**
- Check Console (bottom) for errors
- Make sure you clicked the RIGHT file to run
- HTML projects: run `index.html`
- React/Vue: run `npm start` in Terminal
- Python: run `python main.py` in Terminal

---

### **Issue 4: Can't find my project**

**Fix:**
- Click the **Project dropdown** (top bar)
- Select your project from the list
- Files will load

---

### **Issue 5: Files are in wrong folders**

**Fix:**
- That's how Replit exported them!
- Archetype preserves the exact structure
- You can reorganize files if needed
